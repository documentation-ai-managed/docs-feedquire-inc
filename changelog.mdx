Here's an example webhook payload for payment updates:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Handle incoming webhook
  app.post('/webhook/payments', (req, res) => {
    const { userId, amount, status } = req.body;
    console.log(`Payment of $${amount} for user ${userId}: ${status}`);
    res.status(200).send('OK');
  });
  ```
  ```python
  # Handle incoming webhook
  from flask import Flask, request

  app = Flask(__name__)

  @app.route('/webhook/payments', methods=['POST'])
  def webhook():
      data = request.json
      user_id = data['userId']
      amount = data['amount']
      status = data['status']
      print(f"Payment of ${amount} for user {user_id}: {status}")
      return 'OK', 200
  ```
</CodeGroup>

</Update>
